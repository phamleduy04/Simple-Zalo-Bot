# Simple Zalo Bot

Zalo Bot trả lời theo kịch bản và gửi tin nhắn hàng loạt

## Installation

Cài đặt các packages cần thiết

```bash
npm i
```
Chỉnh sửa lại thông tin trong folder config
## Run
``` bash
npm run start
```

## Admin panel

http://localhost:3000/admin
